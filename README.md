# 📊 Data Mining Project - Retail Analysis

## Project Description

This project analyzes online retail transaction data (2009–2011) to extract insights about customer behavior.

## Objectives

* Customer Segmentation
* Spending Prediction
* Anomaly Detection

## Dataset

Online Retail Dataset (UK-based e-commerce transactions)

## Tools & Technologies

* Python
* Pandas
* Scikit-learn
* Matplotlib

## Techniques Used

* Clustering (K-Means)
* Classification (Decision Tree, Random Forest)
* Regression (Linear Regression)
* Anomaly Detection (Isolation Forest)

1. Open the notebook
2. Run all cells step by step
